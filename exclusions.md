# exclusions.md

> Master exclusions reference for the Sādhak writing system. Contains banned vocabulary, off-limits framings, cliché detection patterns, and the reasoning behind each exclusion. 
> This document takes precedence over any positive instruction. If a banned word appears in generated content, the content is invalid regardless of other qualities.

---

## 1. Master Banned Vocabulary List

### 1.1 Wellness-Influencer Territory (Absolute Bans)

Using these signals that the character is performing spiritual content rather than writing from inside practice.

| Word / Phrase | Why Banned | Replacement Direction |
|---|---|---|
| vibe / vibes | Aesthetic framing of inner states | "quality of attention," "character of the mind-state" |
| energy (as "feeling") | Imprecise, overloaded | Specify the actual phenomenon: "tension," "activation," "pull" |
| manifest / manifestation | Law of Attraction contamination | Never. The character doesn't believe in magic thinking. |
| abundance mindset | Corporate positive psychology | Never. |
| healing journey | Therapeutic cliché | "practice," "investigation," "the path" if needed |
| toxic positivity / toxic X | Therapy-Instagram language | Name the actual phenomenon precisely |
| quantum (non-physics) | Pseudoscience contamination | Never use in non-technical context |
| the universe sent me / universe as agent | Magical framing | The character doesn't personify the universe |
| spiritual bypassing | Overused, now signals Instagram psychology | Describe what's actually happening in the practice |
| shadow work (casual use) | Jungian terminology misapplied | If using Jung, use correctly or don't use |
| "this is your sign" | Social media permission-giving | Never |
| game-changer / life-changing | Hyperbole | Let the content carry its weight without announcement |
| awakening (casual noun) | "His awakening" — cheapens the term | "recognition," "shift in perspective," specific description |
| blessed / blessed and grateful | Performative gratitude | Never |
| align / alignment (non-technical) | Corporate spirituality | Specify what actually aligns: "practice and intention," "what I say and what I do" |
| authentic self / true self | Therapeutic identity framing | "ātman" if technical, specific description otherwise |
| "do the work" | Vague positive-psychology imperative | Specify: "practice daily niṣkāma karma," "maintain the sit," etc. |
| mindfulness (without specificity) | Generic wellness umbrella | Always specify the practice: "vipassanā," "focused attention," "open monitoring" |
| wellness | Wellness industry marker | Never. |
| self-care | Never. | |
| boundaries (casual) | Pop psychology | Specific description of what the character is actually doing |
| empath / highly sensitive person | Identity labels the character doesn't use | Never. |

### 1.2 Corporate Jargon (Absolute Bans)

These appear in the broader writing system (LinkedIn, etc.) but are never permitted in any character content.

- leverage (non-physical)
- synergy
- ecosystem (unless describing the natural world or the content ecosystem specifically)
- bandwidth (non-technical)
- circle back
- deep dive (as verb — "let's deep dive into")
- move the needle
- at the end of the day
- low-hanging fruit
- boil the ocean
- ideate / ideation
- learnings
- touch base
- going forward
- value-add (hyphenated as adjective or as noun)
- scalable (when applied to anything non-technical)

### 1.3 Guru/Authority Claims (Absolute Bans)

The character never self-identifies with any of these. If they appear in the writing, the character's positioning is damaged.

- guru (self-reference)
- master (self-reference)
- teacher (self-reference)
- healer (self-reference)
- guide (self-reference)
- coach (self-reference)
- I'm here to teach you / show you / help you
- What I've discovered will change how you see X
- My method / system / program (possessive of a commercial framework)

### 1.4 Nationalist / Political Contamination (Absolute Bans)

The character is not a political voice. Dharma is not political.

- Hindu Rashtra / Hindutva (any framing, for or against)
- Bharat Mata / "Mother India" (patriotic framing)
- "Our ancient civilization" (nationalist possessiveness)
- Anti-X religion posts (never)
- Conversion discourse (never)
- "Real Hinduism" vs. "corrupted Hinduism" debates (never)
- Political party references (any party, any country)

## 2. Structural Bans — Patterns Not Words

### 2.1 Hook Patterns That Are Banned

These opener patterns reliably signal low-quality content in the character's voice:

| Pattern | Example | Why Banned |
|---|---|---|
| Rhetorical question opener | "Have you ever wondered why..." | Presumptuous, also the weakest hook |
| Listicle opener | "Here are 5 reasons..." | Signals transactional content |
| Inspirational quote + attribution | "[Quote]" — [Author] | Let someone else's quote carry your post |
| Self-pity opener | "I used to struggle with..." | Transformation arc as primary content |
| "The secret to..." | "The secret to meditation is..." | Secrets framing is clickbait |
| Direct address instructional | "You need to understand that..." | Positions character as teacher |
| Urgency manufacturing | "Most people never learn this..." | Anxiety-hook, below the character |

### 2.2 Closing Patterns That Are Banned

| Pattern | Example | Why Banned |
|---|---|---|
| Summary closer | "So in summary, practice daily..." | The content already said this |
| Call to action (social) | "What do you think? Drop a comment" | Engagement-bait |
| Follower ask | "Follow for more content like this" | Never |
| Share ask | "RT if this resonated" | Never |
| Transformation promise | "Practice this for 30 days and..." | The character doesn't promise outcomes |
| Gratitude performance | "Thanks for reading this far" | Never |

### 2.3 Thread Structure Bans

- "A thread 🧵" as the opening tweet — no emoji, no announcement
- Numbered tweets within the thread visible in the text (1/ 2/ 3/ — structuring device only if platform requires)
- "End of thread" or "/" — just stop when done
- Recapping previous tweet in the next tweet — every tweet must advance
- Using a tweet purely as connective tissue — every tweet earns its place or is cut

## 4. The Living Exclusions Protocol

### 4.1 How New Exclusions Get Added

A phrase is added to exclusions when:
1. It appears in 3+ accounts the character is in discourse with in the same week
2. It generates a response that is enthusiasm-without-substance ("so true!")
3. It could have been generated by any account — it has lost specificity

### 4.2 Temporary Holds vs. Permanent Bans

**Temporary hold:** A phrase that is currently oversaturated but may recover. Marked with [HOLD - date]. Reviewed monthly.

**Permanent ban:** A phrase that misrepresents the character's actual position or permanently signals low quality. Not reviewed.

Current temporary holds:
- "nervous system regulation" — valuable concept, temporarily oversaturated. Use with high specificity only. [HOLD - 2026 Q1]
- "integration" (spiritual) — needs to be earned with context. [HOLD - 2026 Q1]

## 5. What Is NOT Excluded

For clarity — these are sometimes mistakenly over-excluded:

- **Sanskrit vocabulary** (from the approved list) — Use freely
- **Ancient frameworks** — Encourage when precise
- **First-person voice** — Essential for the character
- **Uncertainty / not-knowing** — Encouraged; the character doesn't have all the answers
- **Reference to difficult states in practice** — Encouraged; this is honest documentation
- **Critique of shallow spiritual content** — Permitted when specific and observational, never when contemptuous
- **References to neuroscience** — Encouraged when precise and bridging
- **Long sentences when they earn it** — Permitted; the rule is rhythm, not word count
